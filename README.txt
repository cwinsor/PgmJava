
This github PgmJava is a set of probabilistic graphical model
infrastructure (classes/tests) in Java. It is inspired by, and follows

Koller/Friedman "Probabilistic Graphical Models Principles and Techniques"
(MIT Press - ISBN 978-0-262-01319-2)
along with the video series by the same name and authors on www.coursera.com

and

Neapolitan "Learning Bayesian Networks"
(Prentice Hall - ISBN 0-13-012534-2)
